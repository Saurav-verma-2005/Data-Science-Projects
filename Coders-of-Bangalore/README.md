# Coders of Bangalore: Social Media Data Parser

## **Project Overview**
This project demonstrates **Data Engineering and String Manipulation** skills by converting a raw, unstructured text dump of 50+ Instagram profiles into a structured JSON dataset. 

## **Key Challenges Solved**
* **Unstructured Data Parsing**: Created custom logic to handle multi-line text entries with inconsistent lengths.
* **Metric Normalization**: Automated conversion of social media metrics (e.g., "12.5K followers") into standard integers for mathematical analysis.
* **Category Identification**: Aggregated and deduplicated "Type of Page" metadata to identify 34 unique community categories in the Bangalore tech scene.

## **Technical Stack**
* **Language**: Python 3.13
* **Concepts**: String Parsing, Functional Normalization, Data Structuring (JSON)
